# Employee Tracker [![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](https://opensource.org/licenses/MIT)
# Description
This app is a command line application that allows the user to track employees, roles, and departments for a company. The application uses an sql database to track and relate emplyee information. THe application allows the user to add, update, and view employee, role and department information.

# Installation
To use this application clone the repository onto your machine. The user will need node.js and will have to insall all required node packages by running the command "npm i" in the terminal. The user will need to initialize the database a in mysql by running the code in the schema.sql file in the sql workbench. 

# Usage
To start the application run the command "node index.js" fro the the root directory. This will begin the application in the terminal. Follow the menu options and prompts to interact with the application.
